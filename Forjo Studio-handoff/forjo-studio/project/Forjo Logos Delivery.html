Notas de uso

Cuándo usar cuál

El Bauhaus tiene más personalidad — bueno para hero del sitio, presentaciones de portfolio y materiales editoriales. El F Block es más limpio y compacto — ideal para favicon, avatar de redes, stamps en mails y aplicaciones donde la marca queda chica.

Se pueden coexistir: Bauhaus como logo principal de la marca, F Block como sello/avatar secundario.

Para favicon .ico

Bajá el SVG de favicon y convertilo a .ico con realfavicongenerator.net — generá las 6 medidas estándar (16, 32, 48, 96, 144, 192).
